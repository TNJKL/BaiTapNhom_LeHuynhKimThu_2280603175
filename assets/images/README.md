# Hướng dẫn thêm ảnh đại diện

## Ảnh cho Kim Thư (Lê Huỳnh Kim Thư)
1. Lưu ảnh con thỏ với tên: `kimthu_rabbit.png`
2. Đặt file vào thư mục này: `assets/images/kimthu_rabbit.png`
3. Chạy lại app: `flutter run`

## Lưu ý:
- File ảnh nên có định dạng: PNG, JPG, hoặc JPEG
- Kích thước khuyến nghị: 200x200 pixels hoặc lớn hơn
- Tên file phải chính xác: `kimthu_rabbit.png`
